# Programming School

A modern educational web platform designed to help aspiring developers learn programming through structured content, technology-focused learning paths, and engaging user experiences.

The website serves as an online programming school where users can explore programming topics, discover learning resources, read success stories, and access educational content through a clean and responsive interface.

---

## Overview

Programming School was developed as a front-end web project to simulate a real-world educational platform. The project focuses on delivering an intuitive user experience while presenting programming-related learning materials in an organized and visually appealing way.

The platform includes multiple pages that cover different aspects of the learning journey, including course content, user authentication, success stories, and institutional information.
